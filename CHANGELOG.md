# Changelog

## [1.0.0] — Initial Release

### Added
- Full National Pokédex support (Gen I–IX, 1000+ Pokémon)
- 30+ unlockable hints across 4 tiers of strategic value
- Point-based scoring with hint cost, wrong guess penalties, and time factor
- Rank system: Beginner → Master Trainer
- Pokémon cry audio playback
- Silhouette reveal hint
- Pokédex entry (flavor text) hint with Pokémon name hidden
- Scrambled name hint
- Japanese name (kanji + romaji) hint
- Type weakness and resistance charts
- Full stat spread with visual bars
- Autocomplete input from full Pokédex name list
- Evolution method hints (how this Pokémon evolves in and out)
- Responsive layout for mobile and desktop
- Dark theme with Pokémon-inspired colour palette
